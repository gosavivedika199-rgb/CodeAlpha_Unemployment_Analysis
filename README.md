# 📊 Unemployment Analysis with Python

## 📌 Project Overview

This project analyzes unemployment data in India using Python. It includes data cleaning, exploratory data analysis (EDA), and data visualization to understand unemployment trends across different states and over time.

## 🎯 Objectives

- Clean and preprocess the dataset.
- Analyze unemployment rates across different states.
- Study monthly unemployment trends.
- Visualize relationships between numerical variables.
- Generate useful insights from the data.

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📂 Dataset

- Unemployment in India Dataset

## 📊 Visualizations

- Average Unemployment Rate by State
- Monthly Average Unemployment Trend
- Correlation Heatmap

## 📈 Project Insights

- Dataset cleaned successfully by removing missing values.
- Average unemployment rates were calculated for each state.
- Monthly trends highlighted changes in unemployment over time.
- Correlation analysis showed relationships between unemployment rate, employment, and labour participation.


## ▶️ How to Run

```bash
pip install -r requirements.txt
python unemployment_analysis.py
```
